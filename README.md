# Multipass deviantony/docker-elk VM
This project help to create an instance of [deviantony/docker-elk](https://github.com/deviantony/docker-elk) in a multipass vm
## Install VM
### Set parameters:
```
cp env.dist env
```
### Edit env parameters
```
nano env
```
### Install
```
./install.sh
```
### Uninstall
```
./uninstall.sh
```
